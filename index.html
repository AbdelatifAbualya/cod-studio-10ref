How Enhanced CoD Works
Enhanced Chain of Draft uses a revolutionary two-stage API approach:
  • Stage 1: Initial CoD reasoning with deep analytical reflections
  • Stage 2: Critical verification, quality assessment, and answer refinement
  • Result: Higher accuracy through systematic verification and enhancement
  • Cost: ~2x API calls but significantly improved reasoning quality

Controls how many words each Chain of Draft step should target (8-20 words).

Chain of Draft (CoD) is configured to instruct the model to:

  • Use concise CoD steps (target word count adjustable above).
  • Insert 3 "DEEP REFLECTION X:" blocks during thinking (comprehensive analysis).

Available MCP Tools

About MCP Tools

Model Context Protocol (MCP) tools extend the model's capabilities by allowing it to request actions like:

  • Accessing real-time information from the web (e.g., via Tavily).
  • Performing precise calculations (future).
  • Using external systems to retrieve data (future).

Select which tools you want to make available to the model. The model will then decide if and when to use them by outputting a structured request.
